# Odoo 19 Internship Documentation & Transcript Repository

This repository contains structured technical documentation, user manuals, and video transcripts for an Odoo 19 Community Version internship project. It is designed to support both technical and functional learning, with a focus on clarity, accuracy, and Odoo 19 best practices.

## Contents

- **MANUFACTURING/**: Step-by-step documentation and transcripts for 57 Odoo 19 MRP (Manufacturing) training videos, including detailed process flows and functional explanations.
- **odoo-mrp-transcripts/**: Additional transcripts and indexed documentation for Odoo 19 Manufacturing topics.
- **youtube_transcript.py**: Generic Python script to download YouTube video or playlist transcripts for future documentation work.
- **instructions.md**: Agent guidelines and documentation standards for producing Odoo 19 content in this repository.

## Documentation Standards

All documentation in this repository follows these principles:
- Professional, concise, and Odoo 19-specific language
- Structured output: Title, Introduction, Navigation Path, Step-by-Step Procedure, Result Explanation, Conclusion
- Consistent formatting for PDF, Word, and blog publication
- No Enterprise-only features unless explicitly stated
- Quiz-ready answers and terminology alignment with official Odoo 19 docs

## Usage

- Use the provided Python script to generate new transcripts as needed:
  ```
  pip install pytube youtube-transcript-api
  python youtube_transcript.py "<YouTube URL>"
  ```
- Refer to `instructions.md` for documentation style and scope.

## Contribution

This repository is intended for Odoo 19 internship documentation and learning. For contributions, follow the structure and tone described in `instructions.md`.
